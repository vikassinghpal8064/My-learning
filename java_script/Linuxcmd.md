# Important Linux commands which are used mostly

## Print working directory(pwd)
```shell
pwd

```
## list
There are two types of list , you can flag which is useful, ls -l(to see file with extra information) and ls -a(to see extra file like hideen file as well)
```shell
ls -l
ls -a
```

## change directory
```shell
cd ..

```

## who am i (whoami)
```shell
whoami
```

## History
```shell
history
```

## name server look up
it will give the DNS information on the internet
```shell
nslookup domainName
```
